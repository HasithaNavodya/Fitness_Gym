# 🏋️‍♂️ GymFit - Fitness Brand Website

A clean, modern, and fully responsive promotional website for **GymFit**, a local fitness brand. This project was developed as part of the **Web Development Internship Evaluation**.

---

## 🔗 Project Links
* **🌐 Live Demo:** https://fitness-brand-website.vercel.app/
  
* **🎨 Figma Design:** https://www.figma.com/design/wR5a1IGC7dzNgyFBIZQFsv/Fitness-Brand-Website?node-id=0-1&t=N9Q8EkRq5es0LaK2-1

* **💻 GitHub Repository:** https://github.com/HasithaNavodya/Fitness_Brand_Website.git

---

## 🚀 Tech Stack
* **Frontend:** React.js 
* **Styling:** Tailwind CSS 
* **Design Tools:** Figma
* **Version Control:** Git & GitHub 

---

## ✨ Key Features
* **Modern Hero Section:** High-impact landing area to attract new members.
* **About & Services:** Detailed sections showcasing gym services and facilities.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
* **Dark Mode Support (Bonus):** Smooth transition between light and dark themes.
* **Interactive Contact Form:** Includes input validation and success feedback on submission.

---

## 📁 Folder Structure
The project follows a clean and reusable component-based architecture:

```plaintext
src/
 ├── assets/        # Images and logos
 ├── components/    # Reusable React components (Navbar, Hero, Footer, etc.)
 ├── App.js         # Main application logic
 ├── index.js       # Entry point
 └── input.css      # Tailwind CSS configurations
```
## 🛠️ Setup Instructions
Follow these steps to run the project locally on your machine: 

1. **Clone the repository:**
   ```sh
   git clone https://github.com/HasithaNavodya/Fitness_Brand_Website.git
   
2. **Install dependencies:**
   ```sh
   npm install

3. **Start the website:**
   ```bash
   npm start
   ```
   *The app will automatically open at http://localhost:3000 to view it in your browser*.
